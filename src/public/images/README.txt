# Logo Directory

## Instructions for Deployment:

1. Export your GenuVerity logo from Figma Make:
   - Right-click on the logo in the preview
   - Select "Save Image As..."
   - Save as: genuverity-logo.png

2. Place the exported logo file here:
   /public/images/genuverity-logo.png

3. The code in Navigation.tsx and Home.tsx will automatically use this file

## File Requirements:
- Filename: genuverity-logo.png
- Format: PNG (with transparency if needed)
- Recommended size: 400px wide minimum for retina displays
- Location: /public/images/genuverity-logo.png
